# WattLink

**Secure Remote Session Client for WattCoin Merchant Marketplace**

WattLink is a lightweight portable app that enables workers to securely access your PC for remote service jobs (diagnostics, optimization, cleanup, driver updates) through the WattCoin Merchant Marketplace.

## Download

**[⬇️ Download WattLink.exe (Latest)](https://github.com/WattCoin-Org/wattlink-releases/releases/latest/download/WattLink.exe)**

## How It Works

1. **Download** WattLink.exe (portable, no install needed)
2. **Run** it — a session code appears (e.g. `WL-8374-KXPM`)
3. **Post a job** on [wattcoin.org/merchant](https://wattcoin.org/merchant) with your session code
4. **Approve actions** — every tool the worker uses requires your permission
5. **End anytime** — kill switch always available

## Features

- 🔒 **Permission gate** — approve every action before it runs
- 📋 **Activity log** — see everything happening in real-time
- 🛑 **Kill switch** — end the session instantly at any time
- 📦 **Proof package** — full audit trail of all actions taken
- ⚡ **Portable** — single .exe, no installation, no admin required

## Trust & Safety

- All actions require explicit approval (or opt-in trust mode)
- Tools are risk-categorized (low/medium/high/critical)
- Before/after system state captured for proof of work
- Session codes expire after 2 hours if unused

## System Requirements

- Windows 10 or later
- Internet connection
